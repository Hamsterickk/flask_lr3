Hamster Royal Hotel — Flask веб-приложение

Сайт бронирования отеля на Flask: регистрация и авторизация, личный кабинет,
виртуальный кошелёк, бронирование номеров, система отзывов с рейтингом.
Дизайн реализован по макету из Pixso.

Стек

- Backend:Python, Flask, Flask-Login (аутентификация и сессии), Flask-WTF (формы, валидация)
- Frontend:HTML, Jinja2, CSS, GSAP (анимации)
- Хранение данных:сериализация в файл через `pickle`
- Тестирование API:Postman, Swagger / Flasgger (OpenAPI 2.0)

Возможности

- Регистрация и вход, сессии на Flask-Login
- Редактирование профиля
- Виртуальный кошелёк: пополнение картой (валидация номера, срока, CVC), история транзакций
- Бронирование номеров с проверкой баланса и дат заезда/выезда
- Отзывы с рейтингом 1–5
- Серверная валидация всех форм (email, телефон, даты, длины полей)

Тестирование

Каждый эндпоинт покрыт тест-кейсами и проверен вручную в Postman, для API дополнительно
поднята интерактивная документация Swagger (Flasgger) с описанием схем запросов и кодов
ответов (200/201/400/401/404).

По итогам тестирования по тест-кейсам найдены и исправлены реальные баги, например:
- инверсия логики сохранения карты в кошельке (чекбокс работал в обратную сторону);
- отсутствие серверной валидации телефона и дат бронирования;
- дублирующиеся email-поля в модальных окнах профиля, ломавшие вёрстку.

Подробный список изменений — в [`CHANGELOG_PATCH.md`](./CHANGELOG_PATCH.md).

Запуск

pip install -r requirements.txt
flask run

Сервер поднимается на порту 5002 (см. `.flaskenv`).
